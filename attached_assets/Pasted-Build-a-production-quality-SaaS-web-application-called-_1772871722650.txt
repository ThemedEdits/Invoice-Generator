Build a production-quality SaaS web application called “InvoiceForge”.

This is a custom invoice template builder and generator.

The system allows users to upload their own A4 invoice template, visually mark where fields should appear, name those fields, and later generate invoices where the values automatically fill into the exact positions on the template.

The application must feel modern, handcrafted, and professional, not like a typical AI-generated admin dashboard.

Avoid generic UI kits that look like boilerplate.

Design inspiration should be similar to:

Stripe dashboard UX

Linear interface

Figma editor interactions

The app should feel like a real SaaS product with polished UX.

CLOUDINARY STORAGE
Use Cloudinary for storing uploaded invoice templates and generated PDFs.

Cloudinary details:

Cloud name:


dhbwpnsc4
Upload preset:


invoice-generator
The preset is unsigned, meaning uploads can be done directly from frontend.

Upload endpoint:


https://api.cloudinary.com/v1_1/dhbwpnsc4/upload
Upload logic:

Send a POST request with:


file
upload_preset = invoice-generator
Example upload request:

JavaScript

const formData = new FormData();
formData.append("file", file);
formData.append("upload_preset", "invoice-generator");

fetch("https://api.cloudinary.com/v1_1/dhbwpnsc4/upload", {
  method: "POST",
  body: formData
});
The returned secure_url must be saved in Firestore.

6 Smart Calculated Fields
